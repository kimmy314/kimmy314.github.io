Section Analysis Tool

Tool I designed and developed to increase the efficiency of analyzing sections.

This tool was created in Python based using MVC and OOP practices.

The tool takes in simple polygons and calculates the principal moments of inertia along with the theta relative to the graph x axis. The tool also is able to calculate stresses and locate the minimum and maximum stress points for a given load. This allows for users to quickly identify the critical locations and determine if a section fails based on an input load.

A little bit about the design details. An abtract shape class has fields which contain geometry details such as Xcg. The class calculates generic CG positions as well and declares a calculateMOIAboutCG function and defines a calculateMOIAboutZed

The calculateMOIAboutZed is defined within shapes since once a moment of inertia about the CG is found, the MOI about the 0,0 position is simply IXX + A_total * Xcg ^ 2. calculateMOIAboutCG on the other hand is delegated to the subclass Rectangle, Circle and Polygon. Although each shape's MOI could be calculated based off poly's method, it is far more efficient to have Rectangle and Circle separate where they can call an O(1) method for calculating versus poly's O(n) method where n is the number of vertices.
